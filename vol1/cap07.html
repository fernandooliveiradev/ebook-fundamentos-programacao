Capítulo 7

Computação Desplugada

Computação desplugada mostra que computação não é só tela. Antes da máquina, existem representação, regra, sequência e raciocínio.

Computação Desplugada é uma abordagem educacional que ensina conceitos fundamentais de computação sem a necessidade de dispositivos eletrônicos. É particularmente útil em ambientes com recursos limitados ou como complemento ao aprendizado com tecnologia.

Atividade 1: Classificação por algoritmo

Uma atividade clássica de computação desplugada é organizar um baralho de cartas em ordem. Existem várias estratégias (algoritmos) para fazer isso:

  • Selection Sort (seleção): encontrar a menor carta, colocá-la na primeira posição, repetir com o restante.
  • Insertion Sort (inserção): pegar uma carta por vez e inseri-la na posição correta entre as já organizadas.
  • Bubble Sort (bolha): comparar pares adjacentes e trocar se estiverem fora de ordem, repetir até organizar tudo.

Cada estratégia resolve o mesmo problema, mas com eficiência diferente. Com 10 cartas, todas funcionam bem. Com 1.000 cartas, a diferença é enorme — exatamente como na programação.

Atividade 2: Enviar uma mensagem sem tecnologia

Como você comunicaria uma mensagem a alguém do outro lado da sala sem falar, escrever ou usar tecnologia? Uma solução é usar código binário com as mãos:

// Código binário com dedos Dedo levantado = 1 Dedo abaixado = 0 // Uma letra por vez, usando 5 dedos: A = 00001 B = 00010 C = 00011 ... // Isto é um algoritmo de comunicação!

Esta atividade ensina que computadores, no fundo, só entendem 0 e 1 — ligado e desligado. Toda a complexidade que vemos nas telas é construída sobre essa base simples.

Atividade 3: Bebras Brasil

O Bebras é um desafio internacional de pensamento computacional, presente em mais de 80 países. No Brasil, a 5ª edição ocorrerá em 2026 (provas de 01 a 07 de outubro). As tarefas do Bebras são projetadas para serem resolvidas sem computador, usando apenas lógica e raciocínio.

🌐 Participe!

As tarefas do Bebras Brasil estão disponíveis em bebrasbrasil.com.br. São excelentes exercícios de pensamento computacional para todas as idades.

📝 Exercício 1.5 — Computação Desplugada

Com um baralho (ou cartões numerados de 1 a 10), execute o Bubble Sort manualmente:

  1. Embaralhe as cartas e coloque-as em linha
  2. Compare as duas primeiras: se a primeira for maior, troque-as de posição
  3. Avance uma posição e repita a comparação
  4. Continue até o final da linha, depois recomece do início
  5. Repita até que nenhuma troca seja necessária

Quantas passadas completas foram necessárias? Você acabou de executar um algoritmo real sem computador!
