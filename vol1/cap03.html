Capítulo 3

Reconhecimento de Padrões

Reconhecimento de padrões é a habilidade de identificar similaridades, repetições e tendências dentro de um problema — ou entre problemas diferentes. Ao reconhecer padrões, podemos reutilizar soluções e simplificar problemas complexos.

💡 Perguntas-chave

"Já vi algo parecido antes?" • "O que se repete?" • "Como este problema é similar a outro que já resolvi?"

Quando decompomos um problema, frequentemente encontramos padrões entre os subproblemas. Esses padrões — similaridades ou características comuns — podem ser explorados para alcançar soluções mais eficientes. Na literatura acadêmica, o reconhecimento de padrões também é associado ao termo generalização.

Exemplos em diferentes áreas

ÁreaPadrão reconhecido
EconomiaTendências de mercado e comportamentos de consumidores para planejamento estratégico
MeteorologiaPadrões climáticos para elaborar previsões do tempo mais precisas
BiologiaPadrões de migração de espécies e disseminação de doenças
MedicinaSintomas comuns que indicam diagnósticos específicos (gripe vs. infecção bacteriana)
EducaçãoDificuldades recorrentes dos alunos em certos tipos de questões
CozinhaBeijinho e brigadeiro seguem o mesmo padrão de cozimento, apesar de ingredientes diferentes

Padrões na programação

Na programação, reconhecer padrões significa identificar trechos de código que se repetem e transformá-los em funções reutilizáveis. Por exemplo:

// Padrão: calcular média de N números // Se você já resolveu "média de 3 números": media = (n1 + n2 + n3) / 3 // E precisa de "média de 5 números", o padrão é o mesmo: media = (n1 + n2 + n3 + n4 + n5) / 5 // Padrão reconhecido: somar todos e dividir pela quantidade // Isso se torna uma função reutilizável!

Quanto mais padrões são identificados, mais dinâmica e rápida se torna a resolução de novos problemas. Um algoritmo projetado para um problema específico pode ser adaptado para uma série de situações similares, aplicando uma solução generalizada.

📝 Exercício 1.2 — Reconhecimento de Padrões

Identifique o padrão comum nos seguintes problemas e escreva uma solução generalizada:

  • Calcular a média de 3 notas
  • Calcular a média de 5 notas
  • Calcular a média de 10 notas
Ver solução

Padrão: Somar todas as notas e dividir pela quantidade de notas.

Solução generalizada: média = soma_de_todas_as_notas ÷ quantidade_de_notas

Este padrão funciona para 3, 5, 10 ou qualquer quantidade de notas. Na programação, isso se torna uma função reutilizável — um conceito que veremos no Volume 2.
