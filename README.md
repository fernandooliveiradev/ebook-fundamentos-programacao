# 📚 Pensamento Computacional, Lógica de Programação e Algoritmos — E-book Completo

> Pensamento Computacional, Lógica de Programação e Algoritmos

E-book educacional em HTML/CSS, dividido em 3 volumes com 27 capítulos paginados.

## 📖 Conteúdo

### Volume 1 — Pensamento Computacional (7 capítulos)
- História (Papert → Wing), os 4 pilares (decomposição, padrões, abstração, algoritmos)
- Jogos de lógica (Sudoku, Physics Drop)
- Computação Desplugada
- 6 exercícios com soluções

### Volume 2 — Lógica de Programação (10 capítulos)
- Algoritmos, variáveis, operadores, tabelas verdade
- Condicionais (SE/SENÃO/ESCOLHA), laços (PARA/ENQUANTO/FAÇA-ENQUANTO)
- Estruturas de dados (vetores, matrizes, listas, pilhas, dicionários)
- Funções, procedimentos, recursividade
- 35 exercícios com respostas

### Volume 3 — Python na Prática (10 capítulos)
- Instalação (Python 3.14), sintaxe, mapeamento pseudocódigo → Python
- Operadores, condicionais, loops, listas, tuplas, dicionários
- Funções, tratamento de erros (try/except)
- 8 mini-projetos completos
- Recursos para continuar aprendendo

## 🎨 Design
- CSS puro compartilhado (sem frameworks)
- Cor primária azul (#1748e8) — [fernandodeoliveira.pro](https://fernandodeoliveira.pro)
