Capítulo 6

Listas, Tuplas e Dicionários

Até agora, cada variável guardava um valor. Coleções guardam vários valores relacionados, o que permite resolver problemas mais próximos do mundo real.

Imagine uma chamada escolar: não faz sentido criar uma variável para cada aluno. Você cria uma lista de alunos e percorre essa lista quando precisa consultar, contar ou atualizar informações.

Lista

Sequência mutável. Use quando os itens podem entrar, sair ou mudar de posição.

Tupla

Sequência imutável. Use quando o conjunto representa um dado fixo, como coordenadas.

Dicionário

Pares chave-valor. Use quando você quer buscar dados por nome, código ou identificador.

Listas

numeros = [10, 20, 30, 40, 50] print(numeros[0]) # primeiro elemento print(numeros[-1]) # último elemento print(len(numeros)) # tamanho numeros.append(60) numeros.insert(2, 25) numeros.remove(30)

Tuplas

coordenadas = (10, 20) print(coordenadas[0]) # coordenadas[0] = 5 # erro: tuplas são imutáveis

Dicionários

agenda = { "João": "9999-1111", "Maria": "9999-2222", } print(agenda["João"]) agenda["Ana"] = "9999-4444" for nome, telefone in agenda.items(): print(f"{nome}: {telefone}")
Índice começa em zero

No pseudocódigo deste e-book usamos índice começando em 1 em alguns exemplos. Em Python, listas e tuplas começam em 0. O primeiro item é lista[0].

Exercício 3.6 — Coleções

Crie uma lista com cinco notas, calcule a média usando um loop e mostre a maior nota com max().

Coleções armazenam vários dados; funções organizam operações sobre esses dados. No próximo capítulo, vamos transformar blocos úteis em funções reutilizáveis.
