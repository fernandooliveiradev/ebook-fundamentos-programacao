Mini-Projetos Completos

Mini-projetos juntam várias peças em uma solução pequena, completa e testável. É aqui que a teoria começa a ganhar forma de produto.

🚀 Projeto 1: Calculadora Simples

def calculadora(): print("=== Calculadora ===") num1 = float(input("Primeiro número: ")) operacao = input("Operação (+, -, *, /): ") num2 = float(input("Segundo número: ")) if operacao == "+": resultado = num1 + num2 elif operacao == "-": resultado = num1 - num2 elif operacao == "*": resultado = num1 * num2 elif operacao == "/": if num2 != 0: resultado = num1 / num2 else: resultado = "Erro: divisão por zero" else: resultado = "Operação inválida" print("Resultado: ", resultado) calculadora()

🚀 Projeto 2: Jogo de Adivinhação

import random def jogo_adivinhacao(): print("=== Jogo de Adivinhação ===") numero_secreto = random.randint(1, 100) tentativas = 0 while True: palpite = int(input("Adivinhe (1-100): ")) tentativas += 1 if palpite == numero_secreto: print(f"Acertou em {tentativas} tentativas!") break elif palpite < numero_secreto: print("Mais alto! ⬆️") else: print("Mais baixo! ⬇️") jogo_adivinhacao()

🚀 Projeto 5: Verificador de Número Primo

def eh_primo(n): if n < 2: return False for i in range(2, int(n**0.5) + 1): if n % i == 0: return False return True n = int(input("Digite um número: ")) print(f"{n} é primo!" if eh_primo(n) else f"{n} não é primo.")

🚀 Projeto 6: Sequência de Fibonacci

def fibonacci(n): a, b = 0, 1 seq = [] for _ in range(n): seq.append(a) a, b = b, a + b return seq print(fibonacci(int(input("Quantos termos? "))))

🚀 Projeto 8: Calculadora de IMC

def calcular_imc(peso, altura): imc = peso / (altura ** 2) if imc < 18.5: classe = "Abaixo do peso" elif imc < 25: classe = "Peso normal" elif imc < 30: classe = "Sobrepeso" else: classe = "Obesidade" return imc, classe imc, classe = calcular_imc(float(input("Peso (kg): ")), float(input("Altura (m): "))) print(f"IMC: {imc:.1f} — {classe}")

Exercícios integradores de Python

Os exercícios abaixo juntam os quatro blocos da trilha: pensamento computacional, lógica de programação, algoritmos e Python. Em cada um, leia o problema, identifique as partes, pense na regra e só depois compare com a solução.

Exercício 1 — Situação do aluno

Pensamento computacional: separar entrada, cálculo da média, regra de aprovação e saída.

Lógica e algoritmo: se média for maior ou igual a 7, aprovado; se for maior ou igual a 5, recuperação; caso contrário, reprovado.

Python: variáveis, operadores aritméticos, condicionais e função.

Ver solução e resultado
def classificar_aluno(n1, n2, n3): media = (n1 + n2 + n3) / 3 if media >= 7: situacao = "Aprovado" elif media >= 5: situacao = "Recuperação" else: situacao = "Reprovado" return media, situacao media, situacao = classificar_aluno(8, 6, 7) print(f"Média: {media:.1f} - {situacao}")

Resultado esperado: Média: 7.0 - Aprovado

Exercício 2 — Controle de gastos

Pensamento computacional: decompor em lista de gastos, soma, limite e mensagem final.

Lógica e algoritmo: somar todos os valores e comparar com o orçamento disponível.

Python: lista, laço, acumulador e condicional.

Ver solução e resultado
gastos = [120, 80, 35, 60] orcamento = 300 soma = 0 for gasto in gastos: soma = soma + gasto if soma <= orcamento: print(f"Total: R$ {soma}. Dentro do orçamento.") else: print(f"Total: R$ {soma}. Estourou o orçamento.")

Resultado esperado: Total: R$ 295. Dentro do orçamento.

Exercício 3 — Validador de senha

Pensamento computacional: identificar tentativa, senha correta, limite de erros e resultado.

Lógica e algoritmo: repetir enquanto a senha estiver errada e ainda houver tentativas.

Python: while, contador, operador lógico e condicional.

Ver solução e resultado
senha_correta = "python123" tentativas = ["abc", "teste", "python123"] indice = 0 acessou = False while indice < 3 and not acessou: senha = tentativas[indice] if senha == senha_correta: acessou = True indice = indice + 1 if acessou: print("Acesso liberado") else: print("Acesso bloqueado")

Resultado esperado: Acesso liberado

Exercício 4 — Relatório de notas da turma

Pensamento computacional: organizar notas, calcular média, encontrar maior e menor valor.

Lógica e algoritmo: percorrer a lista, acumular soma e comparar valores.

Python: listas, laço, funções prontas e formatação de saída.

Ver solução e resultado
notas = [7.5, 8.0, 6.5, 9.0, 5.0] media = sum(notas) / len(notas) maior = max(notas) menor = min(notas) print(f"Média da turma: {media:.1f}") print(f"Maior nota: {maior}") print(f"Menor nota: {menor}")

Resultado esperado: média 7.2, maior nota 9.0 e menor nota 5.0.

Exercício 5 — Consulta de produto

Pensamento computacional: usar um código como chave para encontrar nome e preço.

Lógica e algoritmo: verificar se o código existe antes de mostrar o produto.

Python: dicionário, chave-valor, if e acesso a dados.

Ver solução e resultado
produtos = { "A10": {"nome": "Caderno", "preco": 18.90}, "B20": {"nome": "Caneta", "preco": 3.50}, "C30": {"nome": "Mochila", "preco": 89.90} } codigo = "B20" if codigo in produtos: produto = produtos[codigo] print(f"{produto['nome']} - R$ {produto['preco']:.2f}") else: print("Produto não encontrado")

Resultado esperado: Caneta - R$ 3.50

Exercício 6 — Caixa eletrônico simples

Pensamento computacional: decompor valor em cédulas maiores primeiro.

Lógica e algoritmo: dividir o valor pela cédula, guardar a quantidade e continuar com o resto.

Python: lista, laço, divisão inteira e módulo.

Ver solução e resultado
valor = 380 cedulas = [100, 50, 20, 10] for cedula in cedulas: quantidade = valor // cedula valor = valor % cedula if quantidade > 0: print(f"{quantidade} cédula(s) de R$ {cedula}")

Resultado esperado: 3 cédulas de 100, 1 de 50, 1 de 20 e 1 de 10.

Exercício 7 — Agenda de contatos

Pensamento computacional: organizar contatos por nome para facilitar a busca.

Lógica e algoritmo: cadastrar, consultar e tratar contato inexistente.

Python: dicionário, função e condicional.

Ver solução e resultado
agenda = {} def adicionar(nome, telefone): agenda[nome] = telefone def buscar(nome): if nome in agenda: return agenda[nome] return "Contato não encontrado" adicionar("Ana", "9999-1111") adicionar("Bruno", "9888-2222") print(buscar("Ana")) print(buscar("Carla"))

Resultado esperado: 9999-1111; depois, Contato não encontrado.

Exercício 8 — Pares e ímpares

Pensamento computacional: reconhecer o padrão: números pares têm resto zero na divisão por 2.

Lógica e algoritmo: percorrer números, testar o resto e separar em duas listas.

Python: operador módulo, listas, laço e condicional.

Ver solução e resultado
numeros = [1, 2, 3, 4, 5, 6] pares = [] impares = [] for numero in numeros: if numero % 2 == 0: pares.append(numero) else: impares.append(numero) print("Pares:", pares) print("Ímpares:", impares)

Resultado esperado: Pares: [2, 4, 6]; Ímpares: [1, 3, 5].

Exercício 9 — Fila de atendimento

Pensamento computacional: entender que o primeiro a chegar deve ser o primeiro atendido.

Lógica e algoritmo: remover sempre o primeiro item da fila até ela ficar vazia.

Python: lista, while e remoção por posição.

Ver solução e resultado
fila = ["Ana", "Bruno", "Carla"] while fila: pessoa = fila.pop(0) print(f"Atendendo {pessoa}") print("Fila encerrada")

Resultado esperado: Atendendo Ana, depois Bruno, depois Carla; por fim, Fila encerrada.

Exercício 10 — Boletim completo

Pensamento computacional: decompor em cadastro de alunos, cálculo de média, regra de aprovação e relatório.

Lógica e algoritmo: para cada aluno, calcular média e classificar situação.

Python: lista de dicionários, função, laço, operadores e condicionais.

Ver solução e resultado
alunos = [ {"nome": "Ana", "notas": [8, 7, 9]}, {"nome": "Bruno", "notas": [5, 6, 5]}, {"nome": "Carla", "notas": [3, 4, 5]} ] def classificar(media): if media >= 7: return "Aprovado" elif media >= 5: return "Recuperação" return "Reprovado" for aluno in alunos: media = sum(aluno["notas"]) / len(aluno["notas"]) situacao = classificar(media) print(f"{aluno['nome']}: média {media:.1f} - {situacao}")

Resultado esperado: Ana aprovada, Bruno em recuperação e Carla reprovada.
