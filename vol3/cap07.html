Capítulo 7

Funções em Python

Funções são blocos nomeados de código. Elas deixam o programa mais legível, reduzem repetição e permitem testar uma parte da solução sem rodar tudo mentalmente.

Do Volume 2 para Python: o FUNCAO do pseudocódigo vira def. O RETORNE vira return.

Função simples

def saudacao(nome): return "Olá, " + nome + "!" mensagem = saudacao("Aluno") print(mensagem)

Parâmetros e retorno

def soma(a, b): return a + b resultado = soma(10, 25) print(resultado)

Parâmetro padrão

def apresentar(nome, idade=18): print(f"Nome: {nome}, Idade: {idade}") apresentar("Carlos") apresentar("Maria", 25)

Função aplicada a uma lista

def calcular_media(notas): total = 0 for nota in notas: total += nota return total / len(notas) notas = [8, 7.5, 9] print(calcular_media(notas))
Nomeie pela intenção

Prefira nomes como calcular_media, validar_idade e formatar_nome. Bons nomes explicam o propósito sem precisar de comentário.

Exercício 3.7 — Função reutilizável

Crie uma função eh_par(numero) que retorne True quando o número for par e False caso contrário.

Funções deixam o programa organizado, mas ainda precisamos lidar com entradas inválidas e situações inesperadas. É aí que entra o tratamento de erros.
