Capítulo 5

Loops em Python

Loops executam o mesmo bloco várias vezes. Eles existem para evitar repetição manual e para lidar com listas, contagens, validações e processos que só terminam quando uma condição muda.

Conexão com algoritmos: repetir também precisa de ordem. Um loop tem começo, condição, corpo e avanço. Se faltar o avanço, ele pode nunca terminar.

while

Use while quando você não sabe exatamente quantas repetições serão necessárias.

contador = 1 while contador <= 5: print(contador) contador += 1

for com range

Use for com range() quando a quantidade de repetições é conhecida.

for i in range(1, 11): print(i) # range(1, 11) gera 1 até 10; o segundo valor é exclusivo.
range()

range(5) gera 0 a 4. range(1, 10, 2) gera 1, 3, 5, 7 e 9. O terceiro argumento é o passo.

for percorrendo uma lista

frutas = ["maçã", "banana", "uva"] for fruta in frutas: print(fruta)

Acumuladores

Um acumulador guarda um resultado parcial. Ele aparece em somas, médias, contagens e relatórios.

total = 0 for numero in range(1, 101): total += numero print(total)
Exercício 3.5 — Repetição

Peça um número e mostre a tabuada dele de 1 a 10 usando for.

Loops ficam ainda mais úteis quando percorrem conjuntos de dados. Listas, tuplas e dicionários dão ao programa uma forma organizada de guardar vários valores e processá-los em sequência.
