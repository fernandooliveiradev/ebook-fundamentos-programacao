Capítulo 5

Loops em Python

while

# while contador = 1 while contador <= 5: print(contador) contador += 1 # equivale a contador = contador + 1

for com range

# for com range (1 a 10) for i in range(1, 11): print(i) # range(1, 11) gera: 1, 2, 3, ..., 10 # O segundo número é EXCLUSIVO (não inclui 11)
💡 range() — atenção ao segundo argumento!

range(1, 11) vai de 1 a 10 (o 11 é exclusivo). range(5) vai de 0 a 4. range(1, 10, 2) vai de 1 a 9 pulando de 2 em 2.

for percorrendo uma lista

# for percorrendo lista frutas = ["maçã", "banana", "uva"] for fruta in frutas: print(fruta)
