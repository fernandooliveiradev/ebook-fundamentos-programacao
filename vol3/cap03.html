Operadores em Python

Operadores são os verbos curtos da linguagem: somam, comparam, combinam condições e transformam valores. Eles são pequenos, mas controlam decisões enormes dentro do programa.

Conexão com o Volume 2: tudo o que você estudou sobre operadores aritméticos, relacionais e lógicos continua valendo. Agora veremos a grafia exata em Python.

Exemplo: par ou ímpar

numero = int(input("Digite um número: ")) if numero % 2 == 0: print("Par") else: print("Ímpar")

Exemplo: aprovação com duas condições

media = float(input("Média: ")) frequencia = float(input("Frequência: ")) if media >= 7 and frequencia >= 75: print("Aprovado") else: print("Rever nota ou frequência")
Atenção

Em Python, = atribui valor e == compara valor. Escreva idade = 18 para guardar e idade == 18 para testar.

Exercício 3.3 — Operadores

Peça um preço e um percentual de desconto. Mostre o valor do desconto e o preço final.

Operadores produzem respostas; condicionais usam essas respostas para escolher caminhos. Sempre que um programa precisa decidir, existe uma comparação sustentando essa escolha.
