Capítulo 2

Sintaxe Básica

Sintaxe é o jeito exato de escrever uma ideia para que a linguagem entenda. A lógica continua a mesma do pseudocódigo; o que muda é a forma de expressar cada passo.

Python foi desenhado para ser legível. Quando o pseudocódigo está bem organizado, a conversão para Python costuma ser direta.

Python troca palavras do pseudocódigo por símbolos e palavras-chave da linguagem.

Variáveis em Python

Em Python, você não precisa declarar o tipo antes. O interpretador infere o tipo a partir do valor atribuído.

# Variáveis em Python — o tipo é inferido automaticamente nome = "Aluno" # str idade = 30 # int altura = 1.75 # float estuda = True # bool print("Olá, " + nome) print("Idade:", idade)

Entrada de dados

A função input() sempre retorna texto. Se você precisa calcular, converta antes.

idade_texto = input("Digite sua idade: ") idade = int(idade_texto) print(f"No ano que vem você terá {idade + 1} anos.")
Indentação importa

Python usa espaços para definir blocos. O recuo não é enfeite: ele diz quais comandos pertencem ao if, ao for, ao while ou a uma função.

Exercício 3.2 — Entrada e saída

Crie um programa que pergunte o nome do aluno e duas notas, converta as notas para número real e mostre a média.

Com variáveis, entrada e saída em Python, já é possível escrever pequenos programas. Operadores ampliam esse repertório, permitindo calcular, comparar e combinar valores.
