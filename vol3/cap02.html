Variáveis em Python (tipagem dinâmica)

# Variáveis em Python — o tipo é inferido automaticamente nome = "Fernando" # str (string) idade = 30 # int (inteiro) altura = 1.75 # float (real) estuda = True # bool (booleano) # Saída print("Olá, " + nome) print("Idade: ", idade) # Entrada entrada = input("Digite seu nome: ")
💡 Indentação importa!

Python usa espaços (indentação) para definir blocos — não usa chaves { } como outras linguagens. Isso torna o código visualmente limpo e obrigatoriamente organizado.
