Condicionais em Python

# if / elif / else nota = 8 if nota >= 9: print("Conceito A") elif nota >= 7: print("Conceito B") elif nota >= 5: print("Conceito C") else: print("Reprovado")

Equivalente ao ESCOLHA/CASO — em Python usamos if/elif:

# Equivalente ao ESCOLHA/CASO dia = 3 if dia == 1: print("Domingo") elif dia == 2: print("Segunda") elif dia == 3: print("Terça") else: print("Dia inválido")
