Capítulo 4

Condicionais em Python

Condicionais (conditionals) permitem que o programa escolha caminhos. Em vez de executar tudo sempre, ele avalia uma condição (condition) e decide qual bloco de código (code block) faz sentido naquele momento.

Pense em uma catraca: se o cartão está válido, libera; se não está, bloqueia. O programa faz esse tipo de decisão com if, elif e else.

if, elif e else

Em Python, if representa o SE, elif representa o SENAO SE e else representa o SENAO. A indentação define quais linhas pertencem a cada bloco.

nota = 8 if nota >= 9: print("Conceito A") elif nota >= 7: print("Conceito B") elif nota >= 5: print("Conceito C") else: print("Reprovado")
Ordem importa

Python testa de cima para baixo. Assim que encontra uma condição verdadeira, executa aquele bloco e ignora os próximos elif do mesmo grupo.

Substituindo ESCOLHA/CASO

Para poucos casos, if/elif resolve bem. O equivalente ao ESCOLHA/CASO (switch/case) do pseudocódigo pode ficar assim:

dia = 3 if dia == 1: print("Domingo") elif dia == 2: print("Segunda") elif dia == 3: print("Terça") else: print("Dia inválido")

Condições aninhadas

Quando uma decisão depende de outra, você pode colocar um if dentro de outro. Use com moderação para não dificultar a leitura.

idade = 17 tem_autorizacao = True if idade >= 18: print("Entrada liberada") else: if tem_autorizacao: print("Entrada liberada com autorização") else: print("Entrada bloqueada")
Exercício 3.4 — Decisão

Crie um programa que leia a temperatura e mostre: “frio” abaixo de 18, “agradável” de 18 a 26 e “quente” acima de 26.

Com condicionais, o programa escolhe caminhos. Com laços (loops), ele repete caminhos até que uma condição seja satisfeita.
