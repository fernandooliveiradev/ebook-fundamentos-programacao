Instalação e Primeiro Programa

Instalando o Python

  1. Acesse python.org/downloads
  2. Baixe a versão mais recente (Python 3.14, lançado em outubro de 2025)
  3. Execute o instalador. Importante: marque a opção "Add Python to PATH"
  4. Abra o terminal e digite: python --version
💡 Versão recomendada

Em 2026, recomendamos Python 3.13 ou superior. O Python 3.14 (lançado em outubro de 2025) é a versão estável mais recente. As versões 3.9 e 3.10 já reached fim de vida.

Seu primeiro programa

Crie um arquivo chamado ola.py e escreva:

print("Olá, Mundo!")

Execute no terminal:

python ola.py
