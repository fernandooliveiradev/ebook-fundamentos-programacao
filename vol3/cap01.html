Capítulo 1

Instalação e Primeiro Programa

Chegou a hora de transformar pseudocódigo em programa executável. A boa notícia: Python é uma das linguagens mais amigáveis para essa transição porque valoriza código limpo e leitura direta.

Antes de instalar: lembre-se do Volume 2. O computador não entende “mais ou menos”. Ele precisa de comandos claros, arquivos no lugar certo e um interpretador capaz de executar esses comandos.

Instalando o Python

  1. Acesse python.org/downloads.
  2. Baixe a versão recomendada para o seu sistema operacional.
  3. No Windows, marque a opção Add Python to PATH no instalador.
  4. Abra o terminal e digite python --version.
Versão recomendada

Em 19 de junho de 2026, a página oficial do Python lista Python 3.14.6, publicado em 10 de junho de 2026, como versão mais recente. Para estudar este e-book, use Python 3.13 ou superior. Python 3.9 está em fim de vida, e Python 3.10 recebe apenas manutenção de segurança até outubro de 2026.

Python lê o arquivo, interpreta as instruções e mostra o resultado no terminal.

Seu primeiro programa

Crie um arquivo chamado ola.py e escreva:

print("Olá, Mundo!")

Execute no terminal, dentro da pasta onde o arquivo foi salvo:

python ola.py
Exercício 3.1 — Primeiro teste

Altere o programa para exibir seu nome, sua cidade e uma frase dizendo por que você está aprendendo programação.

No próximo capítulo, você vai comparar diretamente pseudocódigo e Python. A ideia é perceber que a lógica já foi aprendida; agora mudam a sintaxe e alguns detalhes de execução.
