Funções e Procedimentos

Procedimentos (não retornam valor)

PROCEDIMENTO ExibirCabecalho(titulo: TEXTO) INICIO ESCREVA "========================" ESCREVA " ", titulo ESCREVA "========================" FIM PROCEDIMENTO ExibirCabecalho("Sistema de Vendas")

Funções (retornam valor)

FUNCAO CalcularMedia(n1, n2, n3: REAL): REAL VAR media: REAL INICIO media = (n1 + n2 + n3) / 3 RETORNE media FIM FUNCAO mediaFinal = CalcularMedia(7.5, 8.0, 6.5) ESCREVA "Média: ", mediaFinal

Recursividade

Recursividade é quando uma função chama a si mesma. Exemplo clássico: fatorial.

FUNCAO Fatorial(n: INTEIRO): INTEIRO INICIO SE n <= 1 ENTÃO RETORNE 1 SENÃO RETORNE n * Fatorial(n - 1) FIM SE FIM FUNCAO // Fatorial(5) = 5 × 4 × 3 × 2 × 1 = 120
