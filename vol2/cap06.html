Capítulo 6

Laços de Repetição

Laços de repetição permitem executar um bloco de código múltiplas vezes, evitando repetição manual.

PARA (FOR)

Usado quando sabemos exatamente quantas vezes repetir o bloco.

PARA variavel DE inicio ATE fim PASSO incremento // código a repetir FIM PARA

Exemplo: tabuada:

ALGORITMO Tabuada VAR numero, i: INTEIRO INICIO ESCREVA "Digite um número:" LEIA numero PARA i = 1 ATE 10 PASSO 1 ESCREVA numero, " x ", i, " = ", numero * i FIM PARA FIM

ENQUANTO (WHILE)

Usado quando não sabemos a quantidade de repetições antecipadamente. A condição é verificada antes de cada iteração.

ALGORITMO ValidaSenha VAR senha, tentativa: TEXTO; tentativas: INTEIRO INICIO senha = "1234" tentativas = 0 ENQUANTO tentativa <> senha E tentativas < 3 FAÇA ESCREVA "Digite a senha:" LEIA tentativa tentativas = tentativas + 1 SE tentativa <> senha ENTÃO ESCREVA "Senha incorreta. Tentativa ", tentativas, " de 3" FIM SE FIM ENQUANTO SE tentativa = senha ENTÃO ESCREVA "Acesso liberado!" SENÃO ESCREVA "Acesso bloqueado." FIM SE FIM

FAÇA-ENQUANTO (DO WHILE)

Executa o bloco obrigatoriamente pelo menos uma vez antes de avaliar a condição.

ALGORITMO Menu VAR opcao: INTEIRO INICIO FACA ESCREVA "1 - Cadastrar" ESCREVA "2 - Consultar" ESCREVA "3 - Sair" LEIA opcao ENQUANTO opcao <> 3 FIM
⚠️ Atenção — Loop infinito!

Se a condição do ENQUANTO nunca se torna falsa, o laço executa para sempre! Sempre garanta que algo dentro do laço modifique a condição.

TipoQuando usarVerifica antes?
PARAQuando sabemos o número exato de repetiçõesSim
ENQUANTOQuando a repetição depende de uma condição dinâmicaSim (pode não executar nenhuma vez)
FAÇA-ENQUANTOQuando precisamos executar pelo menos 1 vezNão (executa 1 vez antes de verificar)
📝 Exercício 2.3 — Laços de Repetição

Escreva um pseudocódigo que somar os números de 1 a 100 e mostre o resultado.

Ver solução
soma ← 0 PARA i DE 1 ATE 100 FAÇA soma ← soma + i FIM PARA ESCREVA "A soma é: " + soma

Resultado: 5050 (a famosa soma de Gauss!)
