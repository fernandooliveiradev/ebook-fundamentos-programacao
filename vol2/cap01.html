O que é Algoritmo?

Algoritmo é uma sequência finita de instruções bem definidas para resolver um problema ou executar uma tarefa. Todo programa de computador é, essencialmente, um algoritmo escrito em uma linguagem de programação.

Exemplo do dia a dia: receita de bolo

Pegue a tigela Adicione 2 xícaras de farinha Adicione 1 xícara de açúcar Adicione 3 ovos Misture tudo Leve ao forno por 30 minutos

Exemplo em pseudocódigo: soma de dois números

ALGORITMO SomaDoisNumeros VAR numero1, numero2, resultado: REAL INICIO ESCREVA "Digite o primeiro número:" LEIA numero1 ESCREVA "Digite o segundo número:" LEIA numero2 resultado = numero1 + numero2 ESCREVA "A soma é: ", resultado FIM
💡 Estrutura de um pseudocódigo

Todo pseudocódigo tem: ALGORITMO (nome), VAR (declaração de variáveis), INICIO (corpo), FIM (término). Esta estrutura será mapeada para Python no Volume 3.
