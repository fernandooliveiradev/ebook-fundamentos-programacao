Estruturas de Dados

Vetores (arrays unidimensionais)

Um vetor é uma lista de elementos do mesmo tipo, acessados por um índice (começando em 1 no pseudocódigo).

ALGORITMO MediaNotas VAR notas: VETOR[1..5] DE REAL soma, media: REAL; i: INTEIRO INICIO soma = 0 PARA i = 1 ATE 5 ESCREVA "Digite a nota ", i, ":" LEIA notas[i] soma = soma + notas[i] FIM PARA media = soma / 5 ESCREVA "Média: ", media FIM

Matrizes (arrays bidimensionais)

ALGORITMO MatrizExemplo VAR matriz: VETOR[1..3][1..3] DE INTEIRO; i, j: INTEIRO INICIO PARA i = 1 ATE 3 PARA j = 1 ATE 3 LEIA matriz[i][j] FIM PARA FIM PARA FIM

Listas dinâmicas

lista = [] lista.adicionar(10) lista.adicionar(20) lista.inserir(1, 15) // Resultado: [10, 15, 20] lista.remover(20) // Resultado: [10, 15]

Pilhas (LIFO — Last In, First Out)

Pilha funciona como uma pilha de pratos: o último colocado é o primeiro retirado.

pilha = [] pilha.empilhar("A") pilha.empilhar("B") pilha.empilhar("C") // Estrutura: [A, B, C] topo = pilha.desempilhar() // Retorna "C"

Dicionários (mapas)
