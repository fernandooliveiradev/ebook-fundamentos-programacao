Nível Básico (10 exercícios)

📝 Ex 1 — Par ou Ímpar

Leia um número e informe se é par ou ímpar (use %).

Ver solução
ALGORITMO ParImpar VAR num: INTEIRO INICIO LEIA num SE num % 2 = 0 ENTÃO ESCREVA num, " é par." SENÃO ESCREVA num, " é ímpar." FIM SE FIM
📝 Ex 2 — Maior de Três

Leia três valores e retorne o maior.

Ver solução
ALGORITMO MaiorDeTres VAR a, b, c, maior: REAL INICIO LEIA a, b, c maior = a SE b > maior ENTÃO maior = b FIM SE SE c > maior ENTÃO maior = c FIM SE ESCREVA "O maior é: ", maior FIM
📝 Ex 3 — Contagem Regressiva

Exiba o contador de 10 até 1.

Ver solução
PARA i = 10 ATE 1 PASSO -1 ESCREVA i FIM PARA
📝 Ex 4 — Soma dos Pares

Somar todos os pares de 1 até 100.

Ver solução
soma = 0 PARA i = 1 ATE 100 SE i % 2 = 0 ENTÃO soma = soma + i FIM SE FIM PARA ESCREVA "Soma dos pares: ", soma // 2550
📝 Ex 5 — Tabuada

Exibir tabuada de 1 a 10 de um número.

Ver solução
ALGORITMO Tabuada VAR numero, i: INTEIRO INICIO LEIA numero PARA i = 1 ATE 10 ESCREVA numero, " x ", i, " = ", numero * i FIM PARA FIM
📝 Ex 6 — Média de Notas

Média de 4 notas e validação de aprovação (≥ 7).

Ver solução
ALGORITMO MediaAluno VAR notas: VETOR[1..4] DE REAL; soma, media: REAL; i: INTEIRO INICIO soma = 0 PARA i = 1 ATE 4 LEIA notas[i]; soma = soma + notas[i] FIM PARA media = soma / 4 SE media >= 7 ENTÃO ESCREVA "Aprovado" SENÃO SE media >= 5 ENTÃO ESCREVA "Recuperação" SENÃO ESCREVA "Reprovado" FIM SE FIM
📝 Ex 7 — Fatorial

Calcular o fatorial de N.

Ver solução
ALGORITMO Fatorial VAR n, i, fat: INTEIRO INICIO LEIA n; fat = 1 PARA i = 1 ATE n fat = fat * i FIM PARA ESCREVA "Fatorial: ", fat FIM
📝 Ex 8 — Fibonacci

Exibir N termos da sequência de Fibonacci (0, 1, 1, 2, 3, 5, 8, 13...).

Ver solução
ALGORITMO Fibonacci VAR n, a, b, c, i: INTEIRO INICIO LEIA n; a = 0; b = 1 ESCREVA a, b PARA i = 3 ATE n c = a + b; ESCREVA c; a = b; b = c FIM PARA FIM
📝 Ex 9 — Contador de Vogais

Contar vogais de uma string.

Ver solução
ALGORITMO ContaVogais VAR texto: TEXTO; i, cont: INTEIRO INICIO LEIA texto; cont = 0 PARA i = 0 ATE COMPRIMENTO(texto) - 1 SE CARACTERE(texto, i) = "a" OU CARACTERE(texto, i) = "e" OU CARACTERE(texto, i) = "i" OU CARACTERE(texto, i) = "o" OU CARACTERE(texto, i) = "u" ENTÃO cont = cont + 1 FIM SE FIM PARA ESCREVA "Vogais: ", cont FIM
📝 Ex 10 — Validação de Login

Verificar usuário e senha padrão (use E).

Ver solução
ALGORITMO Login VAR usuario, senha: TEXTO INICIO LEIA usuario, senha SE usuario = "admin" E senha = "1234" ENTÃO ESCREVA "Login realizado!" SENÃO ESCREVA "Usuário ou senha inválidos" FIM SE FIM
