Capítulo 2

Variáveis e Tipos de Dados

Um algoritmo precisa lembrar informações enquanto trabalha. Variáveis (variables) são espaços de memória com nome, valor e significado dentro do problema.

Quando o algoritmo de troco recebe “valor da compra” e “valor pago”, esses dados precisam ficar guardados em algum lugar. Esse lugar é a variável.

Variáveis são espaços na memória que armazenam valores temporários. Pense nelas como caixas com etiqueta: a etiqueta é o nome, e o conteúdo é o valor.

Nome, valor e tipo ajudam o algoritmo a trabalhar com dados corretamente.

Declarando variáveis

No pseudocódigo, declaramos variáveis antes de usá-las. Declarar (declare) deixa claro quais dados participam do algoritmo.

VAR idade: INTEIRO media: REAL nome: TEXTO aprovado: LOGICO

Atribuindo valores

Atribuir é colocar um valor dentro da variável. O valor antigo é substituído pelo novo.

idade = 30 nome = "Ana" media = 8.5 aprovado = VERDADEIRO
Nome de variável importa

Prefira nomes que expliquem o dado: idadeAluno é melhor que x. O computador aceita os dois; quem sofre ou agradece é a pessoa lendo o algoritmo.

Tipos mais comuns

TipoGuardaExemplo
INTEIROnúmeros sem casas decimaisidade = 30
REALnúmeros com casas decimaispreco = 19.90
TEXTOpalavras, frases e caracteresnome = "Ana"
LÓGICOverdadeiro ou falsoativo = VERDADEIRO
Exercício 2.1 — Escolha os tipos

Defina o melhor tipo para: nome de produto, quantidade em estoque, preço, cliente ativo e nota média.

Ver resposta

Nome de produto: TEXTO. Quantidade: INTEIRO. Preço: REAL. Cliente ativo: LÓGICO. Nota média: REAL.

Variáveis guardam valores; operadores transformam esses valores. Depois de saber onde os dados ficam, vamos aprender a somar, comparar e combinar condições.
